Notifications

The tab contains the list of notifications delivered to the user.

Notification filters:
  • SERVICE - messages about changes in service operation. For example, a feature update.
  • PROJECT - messages on specific projects.
  • WARNING - critical messages. For example, about the unavailability of tools in individual projects due to the revocation of access objects
  • ALL - all messages

Operations on notification objects:
  • MORE - expand the card to display more text
  • SKIP - uncheck that the news card has not been read
  • DELETE - delete notification
