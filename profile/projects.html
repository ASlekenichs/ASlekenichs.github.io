The list of available projects is presented on the Projects tab.
On the tab is a list of your projects, as well as projects, access to which is provided by members of your team.
There are several ways to create a new project:
Import src - import existing code sources Import visual - import existing visual models. Create empty - create a new empty workspace
Project filters:
OWNER - projects created by you TEAM - projects created by other users who have granted access ALL - all projects
