A panel for working with a specific project of the application's digital twin.
Includes a tree of digital twin elements. Scheme of data transformation between domains and tools. Notifications that appear in the process of functioning or setting up the transformation scheme.

The element tree includes the following sections:
  • List of digital twin domains
  • List of used tools
  • List of domain rendering layers and tools
  • List of comments
  • List of team members

    • Double clicking on an element from the Domain/Tool groups puts focus on the element on the Basic layer.
    If there is a selected element on the transformation scheme, the tree displays only: associated domains, associated tools and layers on which this element is represented.
    Being selected on one of the layers, the element automatically gets selected on other layers.

    Actions available from the workspace toolbar.
    • ADD DOMAIN - opens the dialog for adding a new domain
    • ADD TOOL - opens the dialog for adding a new tool
    • ADD LAYER - opens the dialog for adding a new layer
    • ADD COMMENT - opens the dialog for adding a new comment
    • DELETE - deletes the selected item
    • PROPERTIES - opens tool property settings panel
    • PUBLISH - starts the publishing process
