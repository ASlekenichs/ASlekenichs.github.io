A bookmark in the workspace element tree. Contains a list of active tools used in the project.

Clicking on an element brings focus to the domain on each of the layers.
After selecting an element on the Layers tab, only layers remain, this tool is present.

To add a new tool, select ADD TOOL. The drop down menu will show:
  • all available tools if there are no selected domains on the layer.
  • all tools that can be used for the domain selected on the layer.

Important! Multiple tools can be connected to the same domain. For example, several Java source tools connect to the Java domain.
