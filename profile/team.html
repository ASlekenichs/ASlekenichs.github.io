Team

The tab contains your requests to join other projects and requests from other Mash=>App users to join your project

To create a request, click NEW and in the dialog that opens, specify the full path to the project you want to access. After creating a request, it will be displayed in the list of your requests with the NEW status.
As soon as access to the project is granted, it will be available to you on the Projects tab with the ALL or TEAM filters enabled. The request itself will receive the CONFIRMED status. If the request is rejected, it will have a DECLINED status.

Filters your requests:
  • NEW - new requests for which no decision has been made
  • CONFIRMED - confirmed requests
  • DECLINED - rejected requests
  • ALL - all requests

At the bottom of the panel, a list of requests for access to your projects is displayed.

Actions on requests:
  • CONFIRM - confirm the request. After confirming the request, the user will have access to the project to which he requested access
  • DECLINE - Decline the request. After the rejection, the user will see that his request has been moved to the rejected status.

You can remove user access from a project on the workspace page.

Incoming request filters:
  • NEW - new requests for which no decision has been made
  • CONFIRMED - requests confirmed by you
  • DECLINED - requests rejected by you
  • ALL - all requests
