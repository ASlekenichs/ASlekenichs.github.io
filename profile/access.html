Access

The tab contains a list of access rights that are granted by other tools within your projects or within projects.

Access object filters:
  • OWNER - access objects provided by you
  • TEAM - access objects created by other users that are used in your projects
  • ALL - all access objects

Operations on Access Objects
  • DELETE - deletes an access object. Attention! Removing an access disables all projects that use the given access.

The creation of a new access object occurs through the NEW call. An access object is created for only one of the available tools or the data provided by it.
