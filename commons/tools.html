Tools

The list of available tools is presented on the Domains&Tools page. In order to display only domains, you need to set the filter Tool.
The use of tools can be:
  • Free
  • Paid with a free period
  • Paid

To view the purchased and paid instruments, you need on this page:
  • Login
  • Enable filter Only accessed

To connect any of the tools directly to the workspace, select add.
Most likely, a particular type of authorization may be required for a particular tool. Therefore, you must first register in the tool itself.
