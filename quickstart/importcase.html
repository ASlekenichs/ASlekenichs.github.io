To import the selected script, do the following:
  • In the workspace page of the script, call the clone operation.
  • If you are not logged in, the system will ask you to log in.
  • The operation will launch the Script Project Clone Wizard.
  • Fill in a new name
  • Select your repositories and tool links.
  • If you have added access rights to the board/sheet, repository, entire project area to your profile, the system displays the Ok sign. Otherwise, the system will prompt you to log in on the side of the tool itself and issue access rights.
  • Add related domains of interest via add
  • Add related tools via add
