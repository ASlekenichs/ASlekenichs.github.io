Import sources

The Import Wizard is available from your personal profile on the Projects tab.
Run the wizard and:
  • Choose project name
  • Provide a link to the repository. In this case, the provider is recognized automatically.
  • If you have access rights to the repository or the entire project area added to your profile, the system displays the Ok sign. Otherwise, the system will offer to authorize on the side of the repository itself and issue access rights.
  • All the rights you add later are available on the Access tab
  • After you finish selecting the necessary repositories, the system will open the workspace window, which will reflect the dynamics of the import process
  • After import, the File domain and all source code and configuration domains are automatically generated. An example is shown in the figure below. If all files of the imported project are recognized, the Ok sign will light up.
  • Add related domains of interest via add
  • Add related tools via add
